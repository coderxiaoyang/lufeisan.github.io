这篇文章我们来学习go的内建变量类型有哪些。

布尔型和字符串

关键词 bool 用于定义 boolean类型变量，boolean 类型变量的取值只有 true 和 false, 零值(默认值)为 false。 golang 不允许把 boolean 类型转变为数字类型。

goalng 使用 UTF-8作为编码结合，使用 “ 或者 包裹起来的字符就是一个 string 定义多行字符串

Error

该错误类型为go内置的 其他语言没有的

golang没有 try-catch 语句块处理异常，内置了 error 类型用于处理错误

整型

这里的整型又分有符号整数(不加u)和无符号整数(加u)，我们同时也可以指定整型的长度，当没有指定长度时候会根据操作系统指定来，32位操作系统是32位，64位操作系统是64位。uintptr是能存储指针的整型。

golang 不允许类型不同的变量进行运算，例如 int8类型的变量和 int32类型的变量不能进行加法操作
