之前我们在Mac上安装Docker的时候 Docker Machine也是同步安装好的。

使用 Docker Machine可以直接创建一个安装好了Docker的虚拟机。

Boot2Docker ISO是一个安装好Docker的ISO文件，就是根据这个ISO去创建虚拟机的。
